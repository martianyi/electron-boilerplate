# Electron Seed

## 调试

clone本项目至本地,cd到项目目录,运行`npm install`安装依赖, 运行`npm start`查看

## 构建应用

运行`npm run build`命令, 打包好的应用位于dist文件夹下

**Windows 构建警告** 需要一字不差地按照[本页](http://electron.atom.io/docs/v0.36.7/development/build-instructions-windows/)的说明进行构建。
